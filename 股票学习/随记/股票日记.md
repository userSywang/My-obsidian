## 🧠 从日记中提炼的教训

> 以下是从你日记原文中提取的核心认知，与纪律对照，方便持续回顾。

### 关于心态
- **我最大的问题是我自己** — 性格缺陷如贪婪、犹豫、侥幸心理，比技术分析更致命
- **别把股票当翻身的筹码** — 就当存银行价值投资
- **冷静，平常心** — 普涨行情不是自己的眼光选择
- **股票始终是副业** — 主业不能荒废

### 关于操作纪律
| 教训 | 关联纪律 |
| ---- | -------- |
| 忘记平仓原因 → 盈利变亏损 | 短线 5 日强制评估 |
| 上涨最高点加仓 → 太急 | 短线禁止追高超 8% |
| 同板块仓位过重 → 跌时无资金抄底 | 单板块 ≤ 50% |
| 明知英伟达上市会跌，未减仓 → 知行不一 | 止损规则 |
| 涨停后还想多吃一个点 → 贪 | 止盈规则 |

### 关于策略调整
- **2026-04-08**：热点抢不到 → 是否提前埋伏，做短线/波段而非超短线
- **2026-04-28**：一半光迅长持 + 一半工业富联波段 → 稳定策略
- **2026-05-22**：考虑 ETF（纳指 + 消费电子/CPO/半导体）+ 个股 + 备用金

## 📅 日记原文

### 前言
我一直都明白我最大的问题是我自己，一些性格的缺陷让我成为股市中的残疾人。

我明知道该去赚哪些钱，可却妄想掌控所有，不让任何机会错过；
我明知道做一件事的风险，却去妄想会不会发生奇迹；
我明知道慢即是快的道理，却从来没有放慢过自己的脚步；
我明知道时光易逝，却从来没有让自己过得有意义。

---

**股票：** 29400 入场 · 3月20日 充值 10000 元
**当前：** 39400

---

### 2026/4/7
杭电股份本来涨了 3% 可以卖了，脑子抽了没卖。首先忘记自己加仓就是为了平仓，且明明知道已经跌过了五日线，股票也是算追高了，且清明假期后行情一般不好，且打仗还没结束可能还存在利空，唯一不知道的就是股东减持。种种因素结合造就今日可能亏 1k。如果还是没有规范自己的操作还是会因为贪而昏头，迟早要亏更多！！

今日迷茫，不懂如何购买股票，就算买的涨了也是因为量化提前埋伏。

### 2026/4/8
既然热点不能抢到，是否可以提前埋伏，尽量不做超短线，做短线或者波段，提前埋伏下一次的行情龙头，这样不至于每天盯盘？

今日大盘这么红反而亏了，还挺难受。

### 2026/4/10
今日买的两只股票都是两连跌，昨天接近跌停入今天还是跌停。思考目前我自己做的是不是超短线，是不是需要改成短线和波段的形式，且拿相对大点的龙头（工业富联/化工/兆易创新等），这样可以避免自己陷入对股票的市场麻木从而误入一些错误的股票（至少在短时间内）。已经对亏损有点麻木，我觉得这个很危险了。

### 2026/4/13
舒华体育连跌 3 天，今天终于不是跌停，但也出了 2/3。目前股票很迷茫，不知道咋玩。

### 2026/4/28
四月就没管住手，不然直接买工业富联都不至于亏这么多。就算是体彩，感觉也应该只买自己熟悉的地方/或大部分人熟悉的（NBA 和部分足球），不然抄作业也不准。

希望后续股票能再规范性一点，目前初步定义：
- 一半钱用于抄底光迅建仓长持
- 一半用于工业富联的波段操作
- 尽量不做超短线

### 2026/4/30
别太精明，别太鸡贼，别把股票当翻身的筹码，就当做存银行价值投资。

### 2026/5/7
今天还是有些想玩短线的苗头。玩可以但我觉得需要清楚这个股票轮动的原因以及持续的时间，这些应该是至少的。最重要的不要把普涨行情当成是自己的眼光选择。

### 2026/5/8
冷静，平常心，这只是普涨行情你碰巧选中了这个股票，你还不懂得选股原理，不要又突然觉得自己行了。股票仍需在学习且仍然只是副业，甚至不是副业。

### 2026/5/11
今天的操作有问题。特发这种小股有考虑但考虑不多，还是想搏一搏，最后是没亏退的。（参考盘中操作技巧中的强承接：股价下跌但均线不怎么跟，说明强承接）

光迅科技这个操作就挺不合常理，是在上涨高点才加仓，有点贪了，正常应该最少等一天内的相对回调加仓。

### 2026/5/12
目前只盈利了 2000 块，别太上头，注意回撤。

可以的，今天的操作没多大问题。后续可能需要注意什么时候 CPO 回调（虽然今天的可能算回调了）。考虑光模块的股票回撤时应该如何执行操作。

### 2026/5/13
同类型的股票应该不能同时持有太多（且满仓）。目前是以光迅科技主仓，用余钱在别的板块搏一搏，现在反而都在光芯片了，是不太行的。

后续如果光迅涨的话，适当的卖掉 200 股在合适的价位。

### 2026/5/14
今天有卖的想法我觉得是可以的，但忘记了在支撑位 208-209 观望。在这个点卖我觉得是更符合之前的设想的。可惜，每次交易还是容易上头。

最近有点过度沉迷股票和基金，忘记了或者不愿意面对自己还需要为自己的主业做准备。记住，股票始终是副业，甚至只是一种投资，主业还是不能荒废学习。

### 2026/5/15
又交学费了，仓位控制的重要性。今天 CPO 全跌也没有资金抄底，眼睁睁的看着。最重要的是仓位全是 CPO。

仓位管理 + 加仓点的学习。

### 2026/5/19
今日纪律不对劲。首先没有预留主力股（光迅）的加仓位，其次当光迅涨后竟然是上升的最顶点买的，还是太急了。涨停是能多吃一个点，但亏是不止亏 1 个点。最差也应该等下来的时候买。

### 2026/5/20
今天虽然暂时没亏钱，但无形中错过一波操作感觉就是亏了。一只趋势票正常来说应该是一点一点稳步上去的，如果突然加速且没有任何利好面的话，很可能是拉高出货，这个时候适当出一点是可以的。

值得庆幸的是没追高，不像昨天虽然是追高但总体亏的不多。

### 2026/5/21
现在需要改变的是思想，不能总想着卖飞怎么办，也应该想亏到本金怎么办。

昨天明明知道英伟达上市今天会跌，为什么没有做到知行合一？
还是想着：卖飞了怎么办。→ 转变思想：亏到本金怎么办。

加仓的点也不对，也是怕涨了之后不赶趟。

**会防守比会进攻更值得夸奖。**

每一次的教训不记着，下一次只会更惨。

### 2026/5/22
考虑要不要换成 ETF 玩，不用选股相对稳定，且看看自己的选股能力/抄作业能力。以 10w 为例：

等一次大回调就这样买：
- 5w ETF（纳斯达克 1w + 消费电子/CPO/半导体 4w）
- 3w 个股
- 2w 备用金

需不需要把钱都放同花顺管理呢？

### 2026/5/25
目前挺可怕的点，自己还是不清楚，不能够确定股票的涨跌逻辑，或者自己没仔细去看过股票的线图。
