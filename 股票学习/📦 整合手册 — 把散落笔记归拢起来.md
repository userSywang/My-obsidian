# 📦 整合手册 — 把散落笔记归拢起来

> **原则：先丢进去，再慢慢整理。** 不要因为追求完美分类而停止记录。

---

## 🎯 核心流程

```
随手记录 ──→  📥 收件箱  ──→  🔄 定期处理 ──→  分类归档
（零门槛）    （临时堆放）     （周/双周）       （永久位置）
```

**两步走，不打断思路：**

### 第 1 步：往收件箱丢（任何时候）

不必想归类，只管扔进来：

| 内容类型 | 怎么丢 |
| -------- | ------ |
| 一段想法/分析 | 直接在 `收件箱/` 新建笔记，写下来 |
| 截图（K线、财报） | 粘贴到收件箱笔记里，或者拖入 `附件/` |
| 微信/雪球看到的好内容 | 复制文字 → 收件箱建笔记 |
| 随手拍的白板/书页 | 图片拖入 `附件/`，在收件箱记一句话 |

> ✅ **收件箱不需要格式**，标题、时间、内容随便写
> ❌ **不要**在收件箱里纠结分类

### 第 2 步：定期处理（建议每周一次）

打开收件箱，逐条问三个问题：

| 这条笔记是…… | 移到…… |
| ------------ | ------- |
| **某只股票的**分析或跟踪 | `个股笔记/` |
| **一个概念/知识点**（如"头肩顶""ROE"） | `概念卡片/` |
| **一种策略/方法** | `策略研究/` |
| **今天/这周的**总结反思 | `复盘日志/` |
| 已过期/没用的 | 直接删掉 |

处理完后在每条笔记开头加一行：
```
已整理 → [[目标笔记]]
```
然后把笔记内容合并到目标文件中，删除收件箱里的草稿。

## 🧩 概念卡片怎么写

概念卡片 = 一个知识点的**最小完整单元**。可以很短，但一定要**自己的话**写。

例子：`概念卡片/ROE.md`

```markdown
---
tags: [股票/概念]
---
# ROE（净资产收益率）

**定义**：净利润 ÷ 股东权益，衡量公司用股东的钱赚了多少钱。

**我的理解**：
- ROE > 15% 算不错，> 20% 很优秀
- 高 ROE 未必是好事——要看是不是靠高杠杆堆出来的（杜邦分析）
- 茅台 ROE 25%+ → 真正的高质量

**关联**：[[概念卡片/杜邦分析]]、[[贵州茅台（示例）]]
```

**特点**：短小、用自己的话、有链接。

## 🔗 截图管理

1. 截图统一丢进 `附件/`，按日期命名：`20250115_K线_茅台.png`
2. 在笔记中用 `![[20250115_K线_茅台.png]]` 引用
3. 如果 Obsidian 有 **"粘贴时自动存入附件文件夹"** 设置，不用手动搬

设置路径：设置 → 文件与链接 → 附件文件夹 → 填入 `股票学习/附件`

## 🚀 现在就做

1. 把你手机/电脑里散落的股票截图，拖入 `附件/`
2. 打开 `收件箱/`，新建一条笔记，写一句你今天关于股票的想法
3. 写一张概念卡片 — 挑你最熟悉的一个概念试试

**不要一次性搬完。每天搬一点，收件箱清零一次算一次。**
