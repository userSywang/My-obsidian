# 📜 股票纪律

> **入场前慎重，入场后管理仓位。**
> 有仓位后第一时间想如何 **不亏本金** → 然后才是 **如何盈利和止盈**。

---

## 1️⃣ 入场条件

### 短线（1~5日）

| 条件 | 描述 |
| ---- | ---- |
| 大盘 | 当日强势 |
| 板块 | 涨停潮或情绪高 |
| 个股 | 放量突破，位于日内低位 |
| **首仓** | 50~80%（轻仓快进快出） |
| ❌ 禁止 | 追高买入涨幅超 8% 的个股 |

### 波段（1~4周）

| 条件 | 描述 |
| ---- | ---- |
| 大盘 | 趋势向上 |
| 板块 | 连续 2 日领涨 |
| 个股 | 量价配合，位于 5/10 日线附近 |
| **首仓** | 30~50% |
| **加仓** | 盈利 5% 后可加仓至满仓 |

### 长线（1~6月）

| 条件 | 描述 |
| ---- | ---- |
| 行业 | 景气周期向上 |
| 个股 | 基本面强，估值合理 |
| 技术 | 位于均线多头排列 |
| **首仓** | 20~30%（分批建仓） |
| **加仓** | 每下跌 5~8% 补仓一次，最多补 2 次 |

---

## 2️⃣ 仓位管理

| 类型 | 进攻 | 防守 | 观望 |
| ---- | ---- | ---- | ---- |
| **短线** | 50~80%（行情明确时） | 20~30% | 0%（不持过夜） |
| **波段** | 60~80% | 30~50% | 0~20%（仅底仓） |
| **长线** | 60~90% | 30~50% | 20~30%（底仓不轻易动） |

---

## 3️⃣ 止损规则

| 类型 | 最大亏损 | 破位信号 | 操作 |
| ---- | -------- | -------- | ---- |
| **短线** | -3~5% | 跌破当日低点或前日低点 | 当日或次日开盘清仓 |
| | | **跌停** | **无条件清仓** |
| **波段** | -7~8% | 有效跌破 10 日线 → 减 50% | |
| | | 有效跌破 20 日线 → **清仓** | |
| | | **跌停** | **无条件清仓** |
| **长线** | -10~15% | 有效跌破 60 日线 → 减 50% | |
| | | 有效跌破半年线 → **清仓** | |
| | | **基本面变化**（逻辑证伪） | **无条件清仓，不设亏损上限** |

---

## 4️⃣ 止盈规则

### 短线（1~5 日）

| 项目 | 内容 |
| ---- | ---- |
| **触发** | 涨 8~15% 或 持有 3 日未继续创新高 |
| **高位信号** | 量能萎缩 / 上影线明显 / 涨速放缓 |
| **操作** | 直接清仓 或 留 20% 观察 |
| **最长持有** | 5 日强制评估 |

### 波段（1~4 周）

| 项目 | 内容 |
| ---- | ---- |
| **触发** | 涨 20~30% + 相对高位 |
| **高位信号** | 五日线上方停滞 / 大盘板块无新高 / 新高附近有压力 |
| **操作** | 减仓 50%，分批离场 |
| **板块回调时** | 保留底仓 10~20% |

### 长线（1~6 月）

| 项目 | 内容 |
| ---- | ---- |
| **触发** | 涨 50%+ 或 基本面/政策逻辑发生变化 |
| **高位信号** | 季线/半年线明显背离 / 板块整体见顶迹象 |
| **操作** | 分三批减仓（各 30% / 30% / 40%） |
| **逻辑消失时** | **无条件清仓，不论盈亏** |

---

## 5️⃣ 仓位约束

| 规则 | 限制 |
| ---- | ---- |
| 同时持有板块 | ≤ 3 个 |
| 单板块仓位上限 | 总仓位的 50% |
| 单票仓位上限 | 短线 30% / 波段 40% / 长线 50% |
| ❌ 禁止操作 | 跌停日加仓 |
| | 亏损超限后补仓摊低 |
| | 无明确逻辑的追涨 |
| | 同时持有多个强相关板块（风险集中） |

---

## 6️⃣ 小白基础规则

> 从零开始的简单纪律，宁可错过不要做错。

- **日线 MACD 金叉时入场**
- **以 20 日均线（MA20）为界** — 收盘跌破即刻离场，绝不恋战
- **价格站稳 MA20 + 成交量同步放大** → 果断进场
- **本金最重要** — 盈利后以保本为主
- **选择性相信第三方博主** — 保持独立判断
