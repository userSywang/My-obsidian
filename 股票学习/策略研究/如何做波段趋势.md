# 📉 如何做波段趋势

> 波段操作的核心：识别趋势，在回调中买入，在冲高中卖出。

---

## 1️⃣ 最核心的工具：黄线均价线

每条分时图上那条**黄色曲线** = 当天所有成交的平均价格。

### 判断真涨还是假涨

| 场景 | 结论 |
| ---- | ---- |
| 股价快速拉升（+5%），但**黄线几乎没动**在 0 轴附近 | ❌ **诱多** — 主力用少量资金拉高，吸引追高，自己出货 |
| 黄线紧跟着黑色分时线一起向上走 | ✅ **真涨** — 买盘真实，资金持续流入 |

> 🗣️ **口诀**：股价冲高，黄线不动 = 假涨，坚决不进场！

---

## 2️⃣ 竞价阶段怎么看（9:15–9:25）

- 只有 **短线龙头股**（板块领涨股）才值得竞价关注
- 平时每天只涨一两个点的股票 → 不用看竞价，等盘中

---

## 3️⃣ 盘中实战：只看"承接"

**承接** = 股价下跌时，是否有真实买盘在下方接住卖盘。

### 强承接 ✅（非常好的买入信号）

| 步骤 | 表现 |
| ---- | ---- |
| 预期 | 今天应该很强（高开高走） |
| 实际 | 快速下跌到 -3% 左右 |
| 关键 | **黄线没有跟着下跌**，股价很快被拉回 |
| 结论 | **大资金在低位大力承接** → 值得关注 |

### 弱承接（可观察）

| 步骤 | 表现 |
| ---- | ---- |
| 股价下跌 | 黄线跟着一起向下 |
| 但 | 跌到 -2~3% 就止住了，没到预期的 -7~8% |
| 之后 | 像上楼梯：每个低点比前一个高，缓慢震荡上行 |
| 结论 | 买盘不算猛，但已开始逐步承接 |

> 🗣️ **口诀**：
> 跌下去，黄线不跟 → **强承接**（好信号）
> 跌下去，黄线跟着跌，但很快稳住 → **弱承接**（可观察）
