# 📈 波段入场点 — 三种入场信号

> 在上涨趋势里"低点买、高点卖"，最关键就是**什么时候买**。

---

## 1️⃣ 最基本的入场信号

股价开始往上走时：
- **5 日均线**向上穿过其他所有均线 ✅
- **20 日均线**也开始慢慢向上拐头 ✅

> 🔑 就像"火车启动了"，趋势正式开始向上，考虑买进。

---

## 2️⃣ 强庄（主力很猛）的两种买点

| 类型 | 特征 |
| ---- | ---- |
| **第一种** | 股价稳稳沿着 5 日线往上走，每天小阴小阳，非常平稳 |
| **第二种** | 突然连续砸出两根大阴线，从 5 日线砸到 20 日线附近 → **反而是机会**（主力已拉起趋势，不会轻易掉头） |

---

## 3️⃣ 弱庄（主力温和）的买点

- 股价回踩到 **10 日线或 20 日线** 附近
- 关键：**成交量明显缩小**（没人慌着卖）
- 缩量 = 抛压小，主力还在悄悄护盘 ✅

---

## 4️⃣ 两条铁律（不管哪种情况）

| 铁律 | 说明 |
| ---- | ---- |
| **量价配合** | 价格上涨时成交量也要放大（量增价升） |
| | 暴涨暴跌时，必须放出历史最大量（天量）才靠谱 |
| **看周线和月线** | 日线容易骗人，周线/月线上的量和趋势更真实 |

---

## 5️⃣ 主力为什么会"洗盘"？

主力拉升前故意**震仓**（让股价晃来晃去），把：
- 爱做 T 的（今天买明天卖）
- 没耐心的
- 看到跌就慌的

→ 这些"浮筹"吓跑。主力只想带着**拿得住、愿意锁仓**的人一起赚钱。

---

> 💡 **一句话核心**：看趋势启动 + 量价配合 + 拿得住 = 跟上主力节奏
