# 🏗️ 主力运作四阶段

> 吸筹 → 洗盘 → 震仓 → 出货 → 再吸筹（循环）

---

## 1️⃣ 吸筹阶段

| 特征 | 说明 |
| ---- | ---- |
| 位置 | 低位缓慢建仓 |
| 手法 | 尾盘抢筹 → 次日开盘砸盘，反复操作 |
| 技术形态 | 大压单 + 大拖单（"弹钢琴"） |
| 持续时间 | 约三个月 |
| 散户策略 | **不介入，仅观察** |

---

## 2️⃣ 洗盘阶段

不是单纯洗人，而是让持仓者的**成本价同步上升**。

| 特征 | 说明 |
| ---- | ---- |
| 价格 | 小阴小阳缓慢推高（如 8 元 → 8.3 元） |
| 目的 | 所有人成本接近当前价位 → 减少拉升抛压 |
| 技术形态 | 5/10/20/30 日均线收敛 |

---

## 3️⃣ 震仓阶段

主力用剧烈日内振幅制造 **恐惧与贪婪循环**。

### 典型分时走势
```
早盘冲高  ──→  你贪婪
击破成本  ──→  你恐惧
尾盘小V/锤子线拉回 ──→  你安心
↕ 反复 3~5 天或一周
```

- 日内振幅从 3~4% 突然放大到 8~9%
- 目的是把心理脆弱者洗出
- 震掉 99% 的散户（仅剩 1% 不看盘的没被洗出）

---

## 4️⃣ 正仓出货阶段

震仓后进入主升浪前，让所有剩余持仓者成本与主力一致。

| 特征 | 说明 |
| ---- | ---- |
| 首板后 | 回踩或沿均线震荡 |
| 等待 | 均线向上靠拢再启动 |

---

## 核心思维

> 主力将除自己以外的所有人都视为**对手盘**。
> 正仓的核心是**心理战**：先诱导贪婪 → 再制造恐惧 → 小幅回补让你安心离场。

---
